Goal:
Build the workerd-safe Verisure transport layer in `packages/server`.

Scope:

- Implement Web fetch-based VerisureTransport with `APPLICATION_ID: PS_PYTHON` behavior.
- Implement host failover between `https://automation01.verisure.com` and `https://automation02.verisure.com`.
- Preserve Python-inspired behavior for network errors, 5xx, 4xx, `SYS_00004`, rate-limit text, and 429.
- Implement cookie header/set-cookie parsing and merging if not already handled in the domain package.
- Return structured Verisure errors, never raw unclassified exceptions.
