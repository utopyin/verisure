Goal:
Build the application-level services used by RPC handlers and Shortcut REST routes.

Scope:

- Implement credential management service flows: create encrypted credential, delete credential, check connection, request/validate MFA, logout.
- Implement `InstallationService` requiring `CurrentUser | CurrentCredential`, including list installations and set/get default giid.
- Implement `AlarmService` requiring `CurrentUser | CurrentCredential | CurrentInstallation`, including get arm state, arm away, arm home, disarm, and toggle full.
- Implement `DeviceService` requiring `CurrentCredential | CurrentInstallation`, including door/window, climate, smart locks, and smart plugs.
- Resolve alarm mutation code from request payload when provided, otherwise from encrypted credential PIN when present.
- Return DTOs from domain/rpc-contract, not raw Verisure GraphQL payloads.
