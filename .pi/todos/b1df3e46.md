Goal:
Implement credential-scoped Verisure authentication and GraphQL execution services.

Scope:

- Implement `VerisureAuth` requiring `CurrentCredential`.
- Support login with decrypted email/password, refresh via `/auth/token`, trust-cookie login, MFA request, MFA validate, logout, and status updates.
- Store/merge cookies and trust token via `VerisureSessionStore`.
- Conservatively set session expiry around the Python-noted 15 minute cookie lifetime.
- Implement `VerisureGraphQL.execute` that ensures a session, sends batched operations to `/graphql`, parses JSON, and maps GraphQL errors.
- Update credential connection status where appropriate: connected, mfa_required, auth_failed, rate_limited, error.
