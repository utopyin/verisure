Goal:
Implement token lifecycle and shortcut export behavior for iPhone Shortcuts automation.

Scope:
- Implement API token generation, hashing, prefix display, optional TOKEN_PEPPER use, authentication, last-used update, expiry checks, scope checks, and revocation.
- Associate each token with a user and Verisure credential; optionally support allowed giids if the schema includes it.
- Implement `ShortcutExportService` for Toggle Full Alarm and Choose Explicit Alarm Mode.
- Generate plaintext token on demand during shortcut export, persist only the hash/prefix, and return export payload with file/import URL or guided fallback details.
- Record shortcut export audit rows.
- Keep this service usable from dashboard RPC handlers; Shortcut REST auth uses the token authentication pieces.
