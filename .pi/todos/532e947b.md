Goal:
Build `apps/api` as the single backend Worker entrypoint mounting health, Better Auth, RPC, and Shortcut REST.

Scope:

- Implement `worker.ts`, `Http/App.ts`, `BetterAuthMount`, `RpcMount`, `RestApi`, route scope helpers, and centralized REST error mapping.
- Mount `GET /api/health`.
- Mount Better Auth under `/api/auth/*` through BetterAuthService.fetch.
- Mount Effect RPC under `/api/rpc/*` using DashboardRpcs and DashboardHandlersLive.
- Implement Shortcut REST endpoints: status, toggle full, explicit mode.
- Authenticate Shortcut REST with `Authorization: Bearer <token>` and provide CurrentUser/CurrentCredential/CurrentInstallation.
- Bind D1, SendEmail, Durable Object namespace, KV cache if used, rate limit, and Config-backed secrets through Alchemy.
- Ensure API route shape is compatible with `verisure.utopy.sh/api/*` custom domain routing.
