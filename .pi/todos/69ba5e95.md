Read first:
- `INITIAL_PRD.md` Implementation Decisions and Testing Decisions
- `docs/architecture.md` sections: Service model, Error taxonomy, Key design decisions
- `repos/verisure-python/verisure/session.py` for GraphQL payload inspiration

Goal:
Build `packages/domain` as a pure package with no Cloudflare, Better Auth, storage, HTTP, or RPC implementation dependencies.

Scope completed:
- Added pure domain DTOs for installations, alarm state/mutation results, device statuses, event log entries, API token summaries, and shortcut export results.
- Added alarm mode helpers for dashboard modes (`DISARMED`, `ARMED_AWAY`, `ARMED_HOME`) and Shortcut-safe modes (`DISARMED`, `ARMED_AWAY`).
- Added structured Effect `Data.TaggedError` domain errors and classification helpers for HTTP/rate-limit/GraphQL responses.
- Added pure Verisure GraphQL operation builders inspired by `repos/verisure-python/verisure/session.py` for:
  - fetch installations
  - arm state
  - arm away
  - arm home
  - disarm
  - door/window sensors
  - climate sensors
  - smart locks
  - smart plugs
  - single smart plug
  - smart plug state mutation
  - event log with Python default categories
- Added pure cookie helpers for parsing `Set-Cookie`, serializing request `Cookie`, and serializing `Set-Cookie`.
- Updated `packages/domain/src/index.ts` to export the domain modules and `VerisureGraphQL` namespace.

Testing/validation:
- Added colocated tests for exact key GraphQL operation payloads, operation names for device queries, event-log defaults, alarm mode mapping, error classification, and cookie helpers.
- `bun run --filter @verisure/domain test` passes.
- `bun run --filter @verisure/domain check:types` passes.
- Full repo `bun run check:types` passes.
- Full repo `bun run test` passes.

Notes:
- Package remains pure and does not import Cloudflare, Better Auth, TanStack, D1, or server implementation modules.
- GraphQL query strings intentionally mirror the Python source first; later service work can refine fields if Verisure's current API requires it.
